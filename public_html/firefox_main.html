这里是我个人常用的 FIrefox 扩展，仅为方便自己重新安装使用。还需要想办法怎么方便的把自己使用的配置文件一些搞定……

Firefox 3 全选

  • - 拦截广告，提高访问速度。
  • - 方便下载一类文件，多线程下载，速度也不错。
  • - 调试代码的好工具，虽然我写 Web 代码也不多 :-)
  • - 为网站自定义 javascript，直接修改网页代码，比扩展应该好一些。
  • - 测试版必备，可使不兼容的扩展标记为兼容。
  • - RTM 的好搭档。
  • - Twitter 伴侣
  • - 以 Vim 模式操作 Firefox，酷且方便。
  • - 多功能 web 开发工具，我主要用它确定网页布局了。

For China 全选

  • - 访问墙外的 Flickr!
  • - Tor 的好搭档。

For Google 全选

  • - Gmail 新邮件提醒
  • - 拦截广告、自定义按钮、建立书签、网页翻译...

For Window$ 全选

其他 全选

  • - 网络视频下载助手，在线看网络太慢了。
  • - wml wap 网页浏览，Opera 是内建的。
