Basic Usage
-----------

* Speedups::

  $ mtn refresh_inodeprints

* Pulling New Branches::

  $ mtn pull --set-default pidgin.im "im.pidgin.*" # from a working copy
  $ mtn -d <path to your pidgin database> pull --set-default pidgin.im "im.pidgin.*" # from anywhere
